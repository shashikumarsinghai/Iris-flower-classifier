# Iris-flower-classifier
A Machine Learning project built with Python, Pandas, and Scikit-learn to classify Iris flower species. This repository demonstrates data preparation, preprocessing, train-test splitting, and the foundation for model training and evaluation.
A beginner-friendly Machine Learning project using Python, Pandas, and Scikit-learn to classify Iris flower species. This repository demonstrates data loading, dataset exploration, preprocessing, feature and target separation, train-test splitting, and serves as the foundation for building and evaluating classification models. The project is organized in Jupyter Notebook format and is ideal for learning, practice, and portfolio development.
